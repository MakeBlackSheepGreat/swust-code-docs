# Installation

## npm (recommended)

```bash
npm install -g swust-code
```

## From Source

```bash
git clone https://github.com/MakeBlackSheepGreat/swust-code.git
cd swust-code
bun install
bun run --cwd packages/opencode src/index.ts
```

## Requirements

- **Node.js** 18+ or **Bun** 1.3+
- **OS** macOS / Linux / Windows (WSL)
- **Memory** 4GB+ recommended

## Verify

```bash
swust-code --version
swust-code --help
```
