# 安装

## npm 安装（推荐）

```bash
npm install -g swust-code
```

## 从源码构建

```bash
git clone https://github.com/MakeBlackSheepGreat/swust-code.git
cd swust-code
bun install
bun run --cwd packages/opencode src/index.ts
```

## 系统要求

- **Node.js** 18+ 或 **Bun** 1.3+
- **操作系统** macOS / Linux / Windows (WSL)
- **内存** 4GB+ 推荐

## 验证安装

```bash
swust-code --version
swust-code --help
```
